撤销与重做


# 撤销和重做操作的命令

```lua
u               -- 撤销[count]次更改 
:u[ndo]         -- 撤销一次更改
:u[ndo] {N}     --转到改变号{N}之后

U               -- 撤销对最近所修改行所做的一系列更改
                -- 当对其他行进行修改，或删除当前行，将失去对该行的记录，此时无法用U恢复
                -- U自己也可作为更改被撤销
```


```lua
<CTRL-R>        -- 重做 [count] 次被撤销的更改
:red[o]         -- 重做一个被撤销的更改
```
